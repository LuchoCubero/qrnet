# QR Net
Proyecto 1 - Redes  

Required tools:  
  - Python 3.6.7  
  - PIP3   
  - Net Tools (Optional)  
  - Scapy:   
  	$ sudo pip3 install scapy  
	$ sudo apt-get install python-scapy3  
	$ sudo pip3 install ipython3  
  - Dumbnet:  
    	$ apt install libdnet python-dumbnet  # or `brew install --with-python libdnet`  
  - Mesh Networking  
    	$ pip install mesh-networking
